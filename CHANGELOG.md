## [0.9.0] - 2026-01-03

### Added
- Initial release of NIS2 Compliance Scanner
- Comprehensive scanning capabilities for ports, TLS/SSL, HTTP headers, and DNS security
- Multiple report formats: HTML, JSON, and Markdown with evidence collection
- Docker deployment support with optional Grafana dashboards
- Security features: secrets detection, WAF/CDN identification, domain monitoring
- Italian compliance checks: P.IVA, privacy policy, cookie consent
- Prometheus integration for metrics export
- Full documentation site at https://fabriziosalmi.github.io/nis2-public
- Configuration management with YAML support
- Interactive CLI for scanning and reporting
- Traefik reverse proxy with automatic HTTPS
- Docker Compose deployment options
